\chapter{Exemples guidés}

\section{Exemple 1 : poutre simplement appuyée}

Objectif : créer une poutre de portée donnée, appliquer une charge répartie et
vérifier les réactions ainsi que le moment maximal.

\subsection*{Étapes}

\begin{enumerate}
  \item Créer deux nœuds aux extrémités de la poutre.
  \item Ajouter une barre entre les deux nœuds.
  \item Définir une section et un matériau.
  \item Affecter les appuis.
  \item Créer un cas de charge.
  \item Ajouter une charge répartie.
  \item Lancer l'analyse.
  \item Comparer les réactions avec la solution analytique.
\end{enumerate}
